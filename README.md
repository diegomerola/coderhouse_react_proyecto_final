# Proyecto E-Commerce React

Proyecto Final E-Commerce para el curso de React Js en Coderhouse. Programado en React y creado con CRA.

## Screenshots

<img src="./src/images/img0001.jpg">

- Se ingresa al sitio y el usuario puede navegar por diferentes categorias, ver los productos e ir a sus detalles.
- En el detalle de producto se ve el titulo, descripción, foto, precio y stock. Para ingresar un producto al carrito se debe seleccionar la cantidad, clickear en ‘add to cart‘ y luego ‘checkout‘ que navega al carrito .
- Cuando el carrito tenga al menos un producto, se visualizara un listado con todos los productos seleccionados y un resumen de la orden con el precio total. Los productos seleccionados se pueden eliminar de a uno o vaciar el carrito. Se mostraran mensajes confirmando dichas acciones.
- Dentro de Cart, al clickear ‘checkout’, se guarda en la base de datos una orden que
  contiene datos del usuario y los productos seleccionados con su respectivo: id, precio, cantidad y fecha.
- Al finalizar la compra se actualizan los datos en la base y se muestra un mensaje con el número de orden.

## Comenzando 🚀

- Para obtener una copia en tu computador local deberas descargar el archivo .ZIP.
- Para clonar en tu repositorio de GitHub deberás forkear este proyecto.

## Instalación 🔧

Instalar "my-project" con npm:

```bash
  npm install my-project
  cd my-project
  npm start
```
